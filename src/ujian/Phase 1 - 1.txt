Ujian

Materi: Basic

Deskripsi
=========
* Buat program penjumlahan 2 bilangan.

Kebutuhan aplikasi
==================
* Program akan meminta pengguna untuk mengisi bilangan ke-1 dan bilangan ke-2.
* Program akan menampilkan hasil penjumlahan bilangan yang di isi oleh pengguna.

Tampilan
========

Masukkan bilangan ke-1: {Bilangan1}
Masukkan bilangan ke-2: {Bilangan2}
{Bilangan1} + {Bilangan2} = {Bilangan3}

Contoh
======
Masukkan bilangan ke-1: 4
Masukkan bilangan ke-2: 3
4 + 3 = 7
